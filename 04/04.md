## if文


ex1) Turning.java

```java
class TurningIf {
	private static void p(String i) {
		System.out.println(i);
	}
	public static void main(String args[]) {
		if (args.length != 1) {
			p("Please Input one number!");
			System.exit(1);
		} else {
			int a = Integer.parseInt(args[0]);
		}
		if (a == 10) {
			p("a -eq 10");
		} else {
			p("a -ne 10"")
		}
	}
}
```

ex2) Eleseif.java

```java
class Elseif {
	private static void p(String i){
		System.out.println(i);
	}
	public static void main(String args[]) {
		if (args.length != 1) {
			p("Please Input one number!");
			System.exit(1);
		} else {
			int a = Integer parseInt(args[0]);
			if (a < 0) {
				p("a < 0");
			} else if (a > 0) {
				p("a > 0");
			} else {
				p("a == 0");
			}
		}
	}
}
```

## while文

```java
class ControlWhile {
	public static void main(String args[]) {
		int i = 0;
		while (i < 5) {
			System.out.println("i is " + i);
		}
	}
}
```

条件文をdo構文で後置する方法もある

```java
class ContrilDoWhile {
	public static void main(String args[]) {
		int i = 0;
		do {
			System.out.println("i is " + i);
			i++;
		} while (i < 5);
	}
}
```
